Is a repository used to understand the core mechanism of a UNIX like OS and C. The goal is to be able to gain controll over the system.

## Installation

1. Make sure C is installed

```bash
# Linux
sudo apt install GCC
gcc --version
```

2. Clone the REPO

```bash
git clone git@github.com:Lmpkessels/unix-systems.git
```

3. Go into the directory

```bash
cd unix-systems
```
